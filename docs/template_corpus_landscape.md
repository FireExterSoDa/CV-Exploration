## Object Folder Convention

Each corpus object is stored as a folder.

```text
docs/template_corpus/<ID_slug>/

README.md
CV.pdf
showcase/
```

`README.md` records the analysis. `CV.pdf` provides a direct visual preview.

For template objects, `showcase/` stores official display assets and `SOURCE.md` records provenance. `CV.pdf` should use an official PDF when available, or be generated from official images when the source does not provide a PDF.
