# Methodology

## Workflow

```text
Template Corpus
       ↓
Design Features
       ↓
Evaluation Framework
       ↓

```

---

# Design Features

## A. Layout

Describe spatial organization.

Examples:

* Single Column
* Double Column
* Module Ordering
* Page Density

---

## B. Visual

Describe visual presentation.

Examples:

* Typography
* Whitespace
* Alignment
* Color
* Icons

---

## C. Information

Describe information organization.

Examples:

* Publication Priority
* Research Interest Position
* Education Weight
* Project Weight
* Contact Information Position

---

## D. Technical

Describe maintainability.

Examples:

* Modular Design
* BibTeX Support
* Multi-page Extension
* Ease of Maintenance

---

## E. Communication

Describe information transmission ability.

Examples:

* Findability
* Reading Path
* Emphasis
* Compression
* Memorability
